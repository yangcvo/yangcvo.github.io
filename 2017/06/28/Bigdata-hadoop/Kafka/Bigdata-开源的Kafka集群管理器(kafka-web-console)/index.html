Kafka Web控制台需要一个关系数据库。默认情况下，服务器连接到嵌入式H2数据库，不需要数据库安装或配置。请咨询Play！的文档以指定控制台的数据库。支持以下数据库：

  • H2（默认）
  • PostgreSql
  • Oracle
  • DB2
  • MySQL
  • Apache Derby
  • Microsoft SQL Server
