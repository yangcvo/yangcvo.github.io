http load 压力测试

一、http_load简介

基于linux平台的一种性能测工具。以并行复用的方式运行，用以测试web服务器的吞吐量与负载，测试web页面的性能。

  • 优点： 1.基于命令行，简单、易于上手。2.小巧轻便，解压缩后不到100k。3.开源，免费。

  • 缺点：1.仅适用于web页面的性能测试，不适用于访问数据库。2.测试结果分析有限3.平台依赖linux。

二、webbench

webbench是Linux下的一个网站压力测试工具，最多可以模拟3万个并发连接去测试网
站的负载能力。下载地址可以到google搜，我这里给出一个

这个程序更小，解压后不到50K，呵呵

用法：

webbench -c 并发数 -t 运行测试时间 URL

如：

webbench -c 5000 -t 120 http://www.vpser.net

  • A 查看请求
